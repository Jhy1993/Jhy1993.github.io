拉普拉斯近似

在PRML 第6章 高斯过程中有应用.

首先说拉普拉斯近似有什么用?

我们都知道高斯分布有很多良好的性质,便于分析.那么如果一个分布$p(z)$不是高斯的怎么办? 那就要通过拉普拉斯近似来找到一个高斯分布$q(z)$来近似它.
$$
p ( z ) = \frac { 1 } { Z } f ( z )
$$

下面来看具体怎么做?

  1. 找到分布$p(z)$的众数.

  2. 找到以该众数为中心的高斯分布$q(z)$.

1. 找到分布$p(z)$的众数.

因为我们的里用来近似的高斯分布$q(z)$是以原始分布$p(z)$的众数为中心(均值)的. 这里众数是概率分布函数$p(z$ )的取到最大值的$z_0$. 直观理解,概率分布的最大值为$p(z_0)$ ,那么就是代表$z_0$的出现概率最大,那么对这个分布$p$采样,应该采样到最多的$z_0$,就是是众数.

所以找到分布$p(z)$的众数 -> 找到其概率分布函数最大值. 一般找最大值最简单的方法就是求导:一阶导数为0(保证极值),且二阶导<0(保证最大).即
$$
\left. \frac { d f ( z ) } { d z } \right| _ { z = z _ { 0 } } = 0, \
\frac { d ^ { 2 }f(z) } { d z ^ { 2 } } | _ { z = z _ { 0 } } < 0
$$

2. 找到以该众数为中心的高斯分布$q(z)$.

这里插播一点泰勒展开的知识(还是大一学的,都忘完了…)

泰勒展开: 函数$f(z)$ 在$z_0$点的泰勒展开为
$$
f(z)= \sum _ { n = 0 } ^ { \infty } \frac { f ^ { ( n ) } ( z_0 ) } { n ! } ( z - z_0 ) ^ { n } \
=f(z_0)+f’(z_0) + \frac{f’’(z_0)}{2}(z-z_0)^2+….
$$

这里为什么对$\ln f(z)$泰勒展开而不是直接对$f(z)$泰勒展开呢?

原因如下:这里泰勒展开出现了二次形式$(z-z_0)^2$ ,我们要近似的$q$高斯分布,它取log之后刚好也是二次函数! 为了让$p$和$q$的形式尽量接近(这样就只剩匹配系数), 所以我们对$\ln p$做泰勒展开.

对$\ln f(z)$在$z_0$进行泰勒展开(注意$f’(z_0)=0$),代入泰勒展开得到:
$$
\ln f ( z ) \simeq \ln f \left( z _ { 0 } \right) - \frac { 1 } { 2 } A \left( z - z _ { 0 } \right) ^ { 2 } \
A = - \frac { d ^ { 2 } } { d z ^ { 2 } } \ln f \left. ( z ) \right| _ { z = z _ { 0 } } >0.
$$

还原指数形式:
$$
f ( z ) \simeq f \left( z _ { 0 } \right) \exp \left{ - \frac { A } { 2 } \left( z - z _ { 0 } \right) ^ { 2 } \right}
$$

现在,$f(z)$已经变成了很像高斯分布的东西了. 这里实际上已经可以看到均值$z_0$和方差$\sigma^2=\frac{1}{A}$ 了,即$\mathcal{N}(z_0,A^{-1})$,只是还差个系数.通过与标准正态分布进行对照可以得到完整形式的高斯分布,即我们要找的$q(z)$.
$$
q ( z ) = \left( \frac { A } { 2 \pi } \right) ^ { 1 / 2 } \exp \left{ - \frac { A } { 2 } \left( z - z _ { 0 } \right) ^ { 2 } \right}
$$

扩展到多维的形式

前面都是$p(z)$,接下来我们将$z$扩展成M维向量,来求解M维分布$p ( \mathbf { z } ) = f ( \mathbf { z } ) / Z$.

泰勒展开变成了
$$
\ln f ( \mathbf { z } ) \simeq \ln f \left( \mathbf { z } _ { 0 } \right) - \frac { 1 } { 2 } \left( \mathbf { z } - \mathbf { z } _ { 0 } \right) ^ { \mathrm { T } } \mathbf { A } \left( \mathbf { z } - \mathbf { z } _ { 0 } \right)
$$
回归指数形式
$$
f ( \mathbf { z } ) \simeq f \left( \mathbf { z } _ { 0 } \right) \exp \left{ - \frac { 1 } { 2 } \left( \mathbf { z } - \mathbf { z } _ { 0 } \right) ^ { \mathrm { T } } \mathbf { A } \left( \mathbf { z } - \mathbf { z } _ { 0 } \right) \right}
$$
同样的,从一维的形式对照过来.
$$
q ( \mathbf { z } ) = \frac { | \mathbf { A } | ^ { 1 / 2 } } { ( 2 \pi ) ^ { M / 2 } } \exp \left{ - \frac { 1 } { 2 } \left( \mathbf { z } - \mathbf { z } _ { 0 } \right) ^ { \mathrm { T } } \mathbf { A } \left( \mathbf { z } - \mathbf { z } _ { 0 } \right) \right} = \mathcal { N } ( \mathbf { z } | \mathbf { z } _ { 0 } , \mathbf { A } ^ { - 1 } )
$$
总结

推导过程很多,但是实际上求近似的高斯分布$q ( \mathbf { z } )$ , 我们只需要知道众数$\mathbf { z } _ { 0 }$ 和二阶Hessian矩阵即可.

整个拉普拉斯近似work的原因是:根据大数定理,在采样到足够多的额样本之后,后验分布式接近一个高斯分布的.
